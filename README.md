# My Generative Music Scripts

This repository contains my personal implementations of generative music algorithms and techniques.

## Current Projects

- **my_lsystem.py** - My implementation of L-Systems for chord generation

## Setup

1. Create a conda environment:
   ```bash
   conda create -n MusicAIPractice python=3.8 -y
   conda activate MusicAIPractice
   ```

2. Install dependencies:
   ```bash
   pip install music21==8.3.0 tensorflow==2.13.0
   ```

## Usage

Run any script with:
```bash
python script_name.py
```
